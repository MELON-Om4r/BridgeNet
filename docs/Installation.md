# Installation

## With Wally

1. Install [Wally](https://wally.run)
2. Put BridgeNet in the ``wally.toml`` file under ``[dependencies]`` 
3. Run ``wally install``

## Without Wally
1. Get the ``.rbxm`` file from the latest release(tbd add link)
2. Sync manually or drop into studio manually
